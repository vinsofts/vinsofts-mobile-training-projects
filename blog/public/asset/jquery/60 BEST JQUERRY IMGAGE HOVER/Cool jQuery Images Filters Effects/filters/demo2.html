Filters

Filters is a jQuery plugin that allows to filter items using custom animation.

The images in the demo are by LINUZ90 and they are licensed under the Attribution-ShareAlike 2.0 Generic (CC BY-SA 2.0) License.

Demos: Basic CSS3 Basic fade in/out Fixed position Custom CSS3 transition 
$('.filters').filters({
	css3: {
		init: false
	},
	move: {
		easing: 'easeOutBack', // we're using jQuery Easing plugin (please download: http://gsgd.co.uk/sandbox/jquery/easing/)
		duration: 400
	},
	fade: {
		duration: [400, 400]
	}
});
