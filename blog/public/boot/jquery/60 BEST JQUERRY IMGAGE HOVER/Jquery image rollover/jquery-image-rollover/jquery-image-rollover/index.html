This is a demo showing too methods of having a transparent image on top of another image on rollover. This method is perfect for setting up a gallery of thumbnails which expand to larger versions of the images when clicked (such as Fancybox). The first option shows a method using jQuery and the second option uses only CSS3. Visit the tutorial to see the code required and for a free download of this tutorial.
